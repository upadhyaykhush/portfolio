  • Defining and Understanding of CNN

    Understanding of Activation Function and their Types.

      Sigmoid Activation Function

      Sigmoid is mainly used as its value fall between 0 and 1. Whenever, it is the case to predict the probability as an output, Sigmoid is used. The Tanh function also behaves just like Sigmoid Function. But, the range of Tanh function is from -1 to 1 which defers from Sigmoid function. The Softmax Function is used for multiclass classification and is more generalized activation function. Sigmoid tend to vanish the Gradient and so RELU was introduced.

      ReLU Activation Function

      ReLU is the most popular and highly preferred Activation Function. The value ranges from 0 to infinity. But the problem with ReLU Function is, all the negative numbers immediately becomes zero and due to this the ability to fit ir train the model decreases. This issue i resolved in Leaky ReLU function.

      Leaky ReLU Activation Function

      It is an attempt to solve the dead ReLU Problem. The function is defined as f(z) = ay. It ranges from (-infinity, infinity). Generally, value of a = 0.01, when a != 0.01, then it is called Randomized ReLU.

    Training the network from SGD Optimizer.

    There are a few downsides of the gradient descent algorithm, lets say we have 10,000 datapoints and 10 features. So, in effect we will be doing 10000 x 10 = 100,000 computations per iteration. In effect we have 10,000 x 1000 = 100,000,000 computations to complete the algorithm. This is quite large for gradient descent and so, Gradient Descent is slow on large datasets.

    Stochastic in plain terms means 'Random'. In SGD, we calculate gradient by picking a random small part of the observations instead of all of them. It reduces computation time in some cases. SGD randomly picks some data points form the whole data at each iteration ro reduce the calculations.
